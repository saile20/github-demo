# github-demo
A simple demo depository to show basics of git workflow

git clone <link>

git remote -v

git log

git config --global user.name <user name>
git config --global user.email <email>

git add .      --> adds all files to staging area
git add file1.txt file2.txt
git reser <filename> -- removes a file from staging

git branch   --> lists all the braches
git branch <branchName>   --> creates a new branch

git commit -m "comment"    --> allows us to create a commit

git checkout <branch name>    --> switch to a different branch

git show <hash>   --- give extensiive commit info
git show --name-only <hash>  -- gives shorthand commit info
git reflog  --> gives info about all commits on all branches
